1. Introduction

Tetris, an iconic and timeless puzzle game, has captivated generations with its simplicity and addictive gameplay. Originally created by Alexey Pajitnov in 1984, Tetris has become a staple in the world of video gaming. Its easy-to-learn mechanics, coupled with its challenging nature, have made Tetris a favorite among casual and competitive gamers alike. In this user interface (UI) remake, we aim to enhance the classic experience by providing a user-friendly interface, optimizing controls for seamless gameplay, and accessibility for people with movement disorder trough Xbox adaptive controller.

2. Interaction Types

2.1 Keyboard and Mouse

The main way of interacting with the interface and the game itself on desktop. Mainly the interface is interacted with the mouse and the game is played with keyboard.

2.2 Touch

The main way of interacting with the interface and the game on phones. The game going controlled trough swipes and two buttons.

2.3 Controller

This option can be used for both desktop and phone and also allow connecting an Xbox adaptive controller for people with mobility disorders.

3.1 Use Flow

3.2 Use Case

The game opens to the main menu for both the windows and mobile version their layouts being similar with the only exception being the settings button.

But to play the game what we are interested in is the play button. Once pressed it starts a new game changing to the game interface.

When the game ends, or we give up on the current game, we are showed the end game screen.

3.3 Design Goal

The goal of the interface is to be approachable and not get in the way of the player nor distracts him from playing the game.

3.4 Visual Metaphors

We use icons the double vertical lines for the paus menu which, the Jolly Roger(skull and bones) for giving up on the game and on mobile we use the gear icon for the settings menu.

3.5 Visual Affordance

We lower the luminosity of the buttons when hovering over them to indicate that they can be interacted with. As for the game screen we sectioned an area for the interactable buttons, demarcating the area with the color white for maximum contrast with the background.
